# Accessibility baseline

NetAdmin Toolbox targets keyboard-first operation, visible focus indicators, semantic headings and live regions for calculated results. Every new tool must provide a labelled control for each input, an accessible name for every button, and a status or alert region for validation feedback.

The automated baseline checks tool discovery and route metadata. Manual release checks remain required for keyboard traversal, screen-reader announcements, contrast, zoom to 200%, reduced motion and touch targets; see [TESTPLAN.md](TESTPLAN.md).
