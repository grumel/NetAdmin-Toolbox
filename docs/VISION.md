# NetAdmin-Toolbox Vision

## Mission

NetAdmin-Toolbox is a fast, privacy-friendly and fully offline-capable Progressive Web App for network, system, security and development administration tasks.

The application provides reliable calculators, converters, generators and diagnostic helpers without requiring accounts, cloud services, telemetry or external runtime dependencies.

## Product principles

- Offline first
- Open source under the MIT License
- Privacy by default
- No tracking or analytics
- No required backend
- No external JavaScript frameworks
- Modular and maintainable architecture
- Accessible keyboard and screen-reader operation
- Responsive desktop and mobile design
- Deterministic, testable calculations

## Target users

- Network administrators
- System administrators
- IT support teams
- DevOps and platform engineers
- Security engineers
- IT trainees, students and instructors

## Product areas

### Network

IPv4, IPv6, CIDR, VLSM, route summarization, subnet planning, DNS and addressing utilities.

### Cisco

ACL, wildcard, VLAN, interface, routing and configuration helpers.

### Windows

PowerShell, DNS, DHCP, SID, GUID and registry utilities.

### Linux

Permissions, cron, systemd, SSH, firewall and shell helpers.

### Security

Password, hash, certificate, JWT, encoding and inspection tools.

### Developer

JSON, YAML, XML, regular expression, UUID and URL utilities.

## Version 1.0 objective

Version 1.0 is a stable, documented, tested and accessible toolbox suitable for regular production use by administrators.
